# Netflix Mobile App UI/UX Design

A Netflix-inspired mobile application UI/UX design created using Figma.

## Features
- Login Screen
- Home Screen
- Search Page
- Movie Categories
- Downloads Section
- User Profile

## Tools Used
- Figma
- UX Design Principles

## Project Goal
To design a modern, responsive, and user-friendly streaming platform interface.
